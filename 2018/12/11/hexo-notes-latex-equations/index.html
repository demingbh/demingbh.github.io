Hexo Notes - NexT 主题下的 LaTeX 公式设置

Posted on 2018-12-11 | In HexoNotes

NexT 本版：hexo-theme-next-6.6.0
此版本 NexT 内建了渲染支持，不需要额外安装东西。

需要修改的文件

  • themes/next/_config.yml
    打开 math 选项，选择 engine: mathjax、per_page: true;
    mathjax 下面的几个 cdn，NexT 官方是推荐第一个，但是我本地 hexo s 之后，打开页面公式显示正常，但是等十几秒，所有公式就都消失了，变成了红字的错误提示，F12 看了下，正是 cdn 出了问题，换成第二个就正常了。
  • source/_posts 里面的 blog 文件
    由于 next/_config.yml 中设置了 per_page: true，意思是为了提高渲染速度，针对每个 page 决定是否启用 MathJax 渲染，这样就需要在每篇 blog 头部添加 mathjax: true 以打开 MathJax 渲染，设置为 false，或者没有设置，则是关闭 MathJax 渲染。

有些奇怪，我的 Hexo 版本是 3.8.0，这样设置公式就显示正常了，并不需要其他操作了。

  • NexT 的 Github 页面上，给出的教程是要求卸载 hexo-renderer-marked，安装 hexo-renderer-pandoc，但是我这么弄了之后，hexo g 就报 pandoc 错误，换回 hexo-renderer-marked 就正常了。
  • 还有的教程上让安装 hexo-math 插件，但是如果使用了 NexT 主题，也不要了。
